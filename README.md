## 🎮 게임 방식

모든 게임이 같은 뼈대를 공유해요:

1. **시작 선택** — 지역/소속사 등, 성장 환경의 트레이드오프를 고르고
2. **턴제 육성** — 매달 훈련 or 휴식. 컨디션을 무시하면 부상, 매달 랜덤 이벤트
3. **정기 이벤트** — 전국대회/쇼케이스가 이닝별 문자중계·무대 연출로 흘러가요
4. **최종 시험** — 드래프트/데뷔 서바이벌에서 3년의 성과가 갈라져요

### 🎤 더 트레이니의 데뷔 후 활동 (신규!)

- 데뷔조 합류·차기 데뷔조 확정이면 **데뷔 활동 진입** — 매년 컴백 준비(연습 3회) → 컴백 무대(킬링파트 미니게임) → 음방 1위·초동·연말 시상식 결산
- 3년차까지 성장, 8년차부터 하락, 10년 계약 만료. 대상·본상·신인상을 모으고 원할 때 은퇴 → 🏛️ 명예의 전당
- ⚔️ 배틀 아레나(컴백 대결)도 야구와 동일하게 지원

### ⚾ 더 루키의 프로 커리어 (신규!)

- 드래프트에 지명되면 **프로 무대 진입** — 매 시즌 스프링캠프(훈련 3회) → 시즌 시뮬레이션 → 결산
- 젊을 땐 성장, 31세부터 노쇠화. 우승 반지·MVP·골든글러브를 모으고 원할 때 은퇴
- 은퇴하면 **🏛️ 명예의 전당**에 커리어 점수와 함께 영구 기록 (얼굴 아바타 포함)
- **⚔️ 배틀 아레나** — 현역/은퇴 선수로 랜덤 매칭 배틀 + 레이팅 랭킹.
  기본은 봇 매칭(오프라인)이고, `rookie/career.js`의 `MATCH_BUCKET`에
  [kvdb.io](https://kvdb.io) 무료 버킷 키를 넣으면 전 세계 플레이어와 매칭돼요
  (`curl -d 'email=본인이메일' https://kvdb.io` 한 번이면 버킷 생성)

밸런스는 전략별 수천 회 시뮬레이션으로 튜닝했어요 — **열심히 집중 육성한 캐릭터가 확실히 좋은 엔딩**을 받습니다.

## 🚀 실행 방법

정적 사이트라 아무 웹서버로나 열면 돼요.

```bash
python3 -m http.server 8000
# → http://localhost:8000
```

GitHub Pages에 올리면 그대로 배포됩니다 (Settings → Pages → main 브랜치 선택).

## 🛠️ 새 게임 추가하기

1. `rookie/` 또는 `idol/` 폴더를 복사해 새 폴더를 만들고
2. `game.js` 상단의 데이터(시작 선택지, 스탯, 이벤트, 엔딩)를 새 테마로 교체
3. `style.css`에서 `:root` 색상 변수만 바꾸면 테마 완성 (`base.css`가 공용 스타일)
4. 루트 `index.html` 허브에 카드 추가

> 재미로 만든 팬게임입니다. 등장하는 학교/구단/기획사 명칭은 분위기를 위한 것으로 실제와 무관해요 🎈
