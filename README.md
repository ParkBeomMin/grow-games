# 키우기 시리즈 🎮

한 판에 5분, 브라우저에서 즐기는 **인생 육성 시뮬레이션 게임 모음**이에요.
설치도 서버도 없이 순수 HTML/CSS/JS로 돌아갑니다.

## 🎮 게임 방식

모든 게임이 같은 뼈대를 공유해요:

1. **시작 선택** — 지역/소속사 등, 성장 환경의 트레이드오프를 고르고
2. **턴제 육성** — 매달 훈련 or 휴식. 컨디션을 무시하면 부상, 매달 랜덤 이벤트
3. **정기 이벤트** — 전국대회/쇼케이스에서 실력을 증명하면 주목도(팬덤)가 상승
4. **최종 시험** — 드래프트/데뷔 서바이벌에서 3년의 성과가 엔딩으로 갈라져요

밸런스는 전략별 수천 회 시뮬레이션으로 튜닝했어요 — **열심히 집중 육성한 캐릭터가 확실히 좋은 엔딩**을 받습니다.

## 🚀 실행 방법

정적 사이트라 아무 웹서버로나 열면 돼요.

```bash
python3 -m http.server 8000
# → http://localhost:8000
```

GitHub Pages에 올리면 그대로 배포됩니다 (Settings → Pages → main 브랜치 선택).

## 🛠️ 새 게임 추가하기

1. `rookie/` 또는 `idol/` 폴더를 복사해 새 폴더를 만들고
2. `game.js` 상단의 데이터(시작 선택지, 스탯, 이벤트, 엔딩)를 새 테마로 교체
3. `style.css`에서 `:root` 색상 변수만 바꾸면 테마 완성 (`base.css`가 공용 스타일)
4. 루트 `index.html` 허브에 카드 추가

> 재미로 만든 팬게임입니다. 등장하는 학교/구단/기획사 명칭은 분위기를 위한 것으로 실제와 무관해요 🎈
